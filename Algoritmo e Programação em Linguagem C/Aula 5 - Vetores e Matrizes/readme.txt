Variáveis indexadas são variáveis que podem ser acessadas por meio de um índice. Isso é útil para armazenar e acessar grandes quantidades de dados.
Vetores são um tipo de variável indexada que pode armazenar um número fixo de dados do mesmo tipo.
Matrizes são um tipo de variável indexada que pode armazenar um número variável de dados do mesmo tipo.
